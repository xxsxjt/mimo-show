# 工作流详细说明

## 续写闭环："读-写-校-更" 四步循环

每次续写严格按以下流程执行，确保长篇创作的一致性：

### Step 1: 续写前（读）

Agent 自动读取以下文档：

1. **foreshadowing.md** — 活跃伏笔 + 推进中伏笔
   - 确保新章节不遗漏已有伏笔
   - 判断哪些伏笔到了该推进/回收的时机
2. **outline_analysis.md** — 近期大纲 + 篇末状态
   - 确保新章节与已有时间线、场景、人物状态对齐
3. **前一章正文** — 上下文衔接
   - 确保开头与上一章结尾自然过渡

### Step 2: 写章节

- 目标字数：2500-3000字
- 段落约40字，碎片段(≤6字)≤5个/章
- 章节标题3-10字，含核心元素
- 标题发布前必须用 `check_duplicate.py` 检测重复
- 遵守光晕表述规范等文风约束

### Step 3: 写后自检

- 扫描确认0处光晕违规
- 节奏检查：紧张/舒缓/过渡？连续3章紧张须穿插缓冲
- 段落质量硬指标：平均30-50字/段，墙段(>100字)≤3个/章

### Step 4: 续写后（更）

- **更新 foreshadowing.md**：新增本章伏笔、标记已回收伏笔
- **更新 outline_analysis.md**：补本章剧情摘要、更新篇末状态
- **每3章迭代一次**，不一次性规划超过3章

### 每10章

- 全面审查 + 总更新
- 批量上传到番茄小说平台

## 发布自动化流程

```
启动 Edge (--remote-debugging-port=9222)
         ↓
连接 CDP (Playwright connect_over_cdp)
         ↓
┌─ 循环：对每个章节 ──────────────────────┐
│  1. 导航到章节管理页                       │
│  2. 点击"新建章节" → 弹出编辑器           │
│  3. 填写标题 (textarea.fill)             │
│  4. 注入正文 (ProseMirror innerHTML)     │
│  5. 等待自动保存                          │
│  6. 点击"下一步"                          │
│  7. 处理弹窗链：                          │
│     - AI声明弹窗 → 选"是"                │
│     - 风险检测弹窗 → 取消跳过             │
│     - 确认发布弹窗 → 点击确认             │
│  8. 等待跳转到目录页 = 发布成功           │
│  9. 验证：在目录页搜索标题               │
│     - 找到 → 成功，继续下一章             │
│     - 未找到 → 刷新重试                   │
│     - 仍失败 → 停止，等待人工介入         │
└──────────────────────────────────────────┘
```

### 关键技术细节

- **正文注入方式**：不用 keyboard.type 逐字输入（慢且易出错），也不直接 fill()，而是将 Markdown 转为 HTML 段落，通过 JS 直接赋值 ProseMirror 的 innerHTML
- **重复标题检测**：番茄不允许重复章节名，check_duplicate.py 在写前检测
- **每日字数上限**：番茄有当日发布字数上限，超过后"确认发布"无效，次日重置
- **发布后验证**：每章发布后在目录页搜索标题确认，失败则刷新重试

## 多文档协同状态机

```
                    ┌──────────────┐
                    │ characters.md │ ← 角色卡片 + 信息差矩阵
                    └──────┬───────┘
                           │ 涉及角色时查
                           ↓
┌──────────────┐   ┌──────────────┐   ┌──────────────┐
│ foreshadowing │──→│   写章节     │──→│   outline    │
│     .md      │   │  (AI续写)    │   │  _analysis   │
└──────────────┘   └──────┬───────┘   └──────────────┘
     ↑ 写前读               │ 写后更新        ↑ 写后更新
     │                      ↓                 │
     │              ┌──────────────┐           │
     └──────────────│  自检+更新   │───────────┘
        新增/回收伏笔  └──────┬───────┘  补剧情摘要
                           │
                           ↓
                    ┌──────────────┐
                    │  番茄小说     │ ← Playwright CDP 自动发布
                    └──────────────┘
```

### 四份核心文档

| 文档 | 用途 | 更新频率 |
|------|------|----------|
| foreshadowing.md | 伏笔追踪（活跃+推进中） | 每章写后 |
| outline_analysis.md | 近期大纲时间线+篇末状态 | 每章写后 |
| characters.md | 角色卡片+信息差矩阵+当前状态 | 涉及角色时 |
| worldbuilding.md | 世界观设定（境界/天赋/地理/势力） | 涉及设定时 |
