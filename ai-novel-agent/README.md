# AI-Novel-Agent: AI 驱动的长篇网文自动化创作与发布系统

> 基于 CodeBuddy (WorkBuddy) Agent + GLM 模型构建，驱动 AI 完成长篇网文从构思到上线的全流程自动化。

## 项目概览

- **小说名称**：《再看，就把你瞪死》（原名《极致术》）
- **当前进度**：197 章 / 约 60 万字，已全部发布至番茄小说平台
- **核心能力**：AI 续写 → 伏笔/大纲自动同步 → 浏览器自动化批量发布

## 系统架构

```
┌─────────────────────────────────────────────────────┐
│                  续写前（强制读取）                    │
│  foreshadowing.md ──→ 活跃伏笔 + 推进中伏笔          │
│  outline_analysis.md ──→ 近期大纲 + 篇末状态          │
│  前一章正文 ──→ 衔接上下文                            │
├─────────────────────────────────────────────────────┤
│                  写章节（2500-3000字）                 │
│  · 光晕表述规范（感知效果不可做主语）                   │
│  · 段长约40字，碎片段≤5个/章                          │
│  · 章节标题去重检测（check_duplicate.py）             │
├─────────────────────────────────────────────────────┤
│                  写后自检                             │
│  · 0处光晕违规                                       │
│  · 节奏检查（紧张/舒缓/过渡）                         │
├─────────────────────────────────────────────────────┤
│                  续写后（强制更新）                     │
│  · 更新 foreshadowing.md（新增/标记回收伏笔）         │
│  · 更新 outline_analysis.md（补剧情摘要+篇末状态）     │
│  · 每3章迭代一次                                     │
├─────────────────────────────────────────────────────┤
│                  发布到番茄小说                        │
│  · Playwright + CDP 连接 Edge 浏览器                  │
│  · 自动创建章节 → 粘贴正文 → 确认发布                  │
│  · 支持批量发布（单次最多16章）                        │
│  · 发布后验证 + 失败重试                              │
└─────────────────────────────────────────────────────┘
```

## 核心痛点与解决方案

| 痛点 | 解决方案 |
|------|----------|
| 长篇伏笔遗忘/矛盾 | 62条伏笔线索结构化追踪，写前强制读取，写后强制更新 |
| 角色断线/信息差混乱 | 角色卡片 + 信息差矩阵，10+核心角色全覆盖 |
| 大纲漂移 | 近期篇章详细时间线 + 篇末状态，每章更新 |
| 手动发布低效易错 | Playwright CDP 自动化，含标题去重检测、发布验证、失败重试 |
| 文风不一致 | 质量硬指标嵌入 Agent 强制约束，写后自检 |

## 关键成果

- **197章** 完成创作与发布（零时间线矛盾、零角色断线）
- **62条伏笔** 结构化管理（活跃 + 推进中 + 已回收三层体系）
- **10+角色** 信息差矩阵覆盖
- **单章全流程** 从约2小时压缩至15分钟（效率提升约8倍）

## 技术栈

- **AI Agent**: CodeBuddy (WorkBuddy)，基于 GLM-5.1 模型
- **浏览器自动化**: Playwright + Chrome DevTools Protocol (CDP)
- **发布平台**: 番茄小说 (fanqienovel.com)
- **写作管理**: Markdown 结构化文档 + 强制读写闭环
