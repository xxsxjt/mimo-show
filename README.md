# MiMo Show

个人 AI 项目展示。

## 项目

### 1. [ai-novel-agent](./ai-novel-agent/)

用 CodeBuddy Agent + GLM 模型辅助写网文的工具链。核心是几份结构化 Markdown 文档（伏笔追踪、大纲时间线、角色卡片）配合 Agent 的读写闭环，减少长篇写作中伏笔遗忘和情节矛盾的问题。另外写了个 Playwright 脚本自动往番茄小说发布章节。

实际用了大半年，写了约 197 章 / 60 万字。不算成熟产品，更像是一套给自己用的 workflow + 脚本，细节上很多地方还是靠人工兜底。

### 2. [ai_sandbox](https://github.com/xxsxjt/ai_sandbox)

AI 交互实验项目，纯前端为主，接 OpenAI 兼容 API。做了几个模块：多角色对话模拟、狼人杀、战斗模拟、占卜、AI 回复分析、文字冒险游戏。后端是可选的 Java 小服务，主要做联网搜索和跨域代理。

功能都有但比较粗糙，属于边学边做的练手项目。
