# world-guardian（万界守卫）

多人在线塔防生存游戏，用 Claude Code 生成。

## 怎么玩

支持4人同时在线，两种模式（经典/无限生存）。三种职业（建筑师、守卫者、炼金术士），在六边形地图上建塔防守，打怪升级。

## 技术实现

- **服务器**：Node.js + TypeScript + WebSocket，10+ 个游戏系统（战斗、经济、塔防、敌人生成、波次、拍卖、地图、程序化地图、毒圈、守卫者等），模块化架构
- **客户端**：Vue 3 + TypeScript + Canvas，六边形地图渲染，状态插值优化网络同步

## 功能

- 多种敌人类型（基础、快速、坦克、治疗者、分裂者、生成者、潜行、元素护盾、Boss）
- 四种元素系统（火、冰、雷、毒）+ 五种防御塔
- 程序化地图生成，每局不同
- 动态毒圈（无限生存模式）
- 经济系统：灵晶、石头、经验、技能升级

## 实际情况

AI 生成代码大概 5000+ 行，基本功能能跑。但游戏平衡性没怎么调，有些系统交互有 bug，属于 demo 级别。WebSocket 延迟在高频操作时偶有卡顿，没有做生产级优化。

## 技术栈

- 服务器：Node.js / TypeScript / WebSocket (ws)
- 客户端：Vue 3 / TypeScript / Vite / Canvas API / Pinia
